Sven Bakker, Dylan de Metter en Marcel Touwen

In dit project hebben wij een 2d platform game gemaakt en verbeterd. Er zijn 12 levels die steeds in moeilijkheid oplopen.

Jungle Dash
https://github.com/Schooier/game

installatie instucties:
installeer visual studio code
installeer de extenties python en pygame
Zet de mapjes in de python die je aanmaakt in de files
Spelen
open de main.py
klik op run en daarna op run without debugging
nu kan je het spel spelen

Spelen:
Up key is springen
Left key is naar links lopen
Right key is naar rechts lopen
Dit geld ook voor WASD
W is springen
D is naar rechts lopen
A is naar links lopen
P of ESC is op pauze drukken
Op deze manier kan je alle levels doorspelen

Er is maar één bug die soms vervelend is, als je op een bewegend platform staat en die gaat onder een tile door dan sta je automatisch op die tile.
